# College Management System

## Project Overview

The College Management System is a backend-based application developed
to manage and organize college-related data such as Students, Staff,
Books, and Departments.

This project focuses on implementing CRUD operations (Create, Read,
Update, Delete) using MongoDB as the database and backend technologies
without any frontend interface. API testing can be performed using
Postman or Swagger.

------------------------------------------------------------------------

## Objective

The objective of this project is to build a College Management System
that: - Stores and manages college data efficiently. - Performs CRUD
operations on different entities. - Establishes relationships between
collections/tables. - Demonstrates backend development concepts and
database integration.
